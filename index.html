QA Playground

A lightweight sandbox to demonstrate UI automation, workflows, visual regression and error handling for QA engineering.

Explore the test areas

Each page focuses on a specific aspect of testing and can be used as a target for Playwright, Selenium, Robot Framework or API tools.

  • Login Flow – authentication, token storage, redirection.
  • Multi-step Form – step navigation, validation, summary.
  • Dynamic Table – API-driven content, DOM updates, errors.
  • UI Actions – modals, loaders, dynamic messages.
  • Error Scenarios – JS errors, failing and slow APIs.
  • Visual Regression – stable components for screenshots.
